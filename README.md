# cartao_WillianDias

Cartão de apresentação profissional com foco em clareza visual, links diretos e baixo atrito de carregamento.

## Governança

- CI: [./.github/workflows/ci.yml](./.github/workflows/ci.yml)
- Segurança: [./.github/SECURITY.md](./.github/SECURITY.md)
- Pull requests: [./.github/pull_request_template.md](./.github/pull_request_template.md)

## Boas práticas aplicadas

- `rel="noopener noreferrer"` em links externos com `target="_blank"`
- `preconnect` para Google Fonts
- metatags básicas de descrição e tema
- validação automática de links e assets via GitHub Actions
